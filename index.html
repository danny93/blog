Skrypty

Skrypt liczący znaki.

#!/bin/bash
 l1=`cat $1 | wc-m`
 l2=`cat $2 | wc-m`

if[$l1 -gt $l2]
echo "wiecej znakow ma plik $1"
else
echo "wiecej znakow ma plik $2"

1. Skrypt który zmienia nazwę dopisując sufiks w postaci _old

 #!/bin/bash
 

 for nazwa in `ls`
     do
       echo "$0"
       echo "$nazwa"
       if [ $nazwa != `basename $0` ]
       then
       cp $nazwa $nazwa"_old"
       fi
     done
     
exit 0

3. Skrypt z pętlą while(który byl na zajęciach). *Pętla while wykonywana jest tak długo jak długo podany warunek jest prawdziwy.* 

 #!/bin/bash


      i=$1;
      while ((i<=$2));
      do
      echo "$i"
      let "i+=1";
     done

exit 0

4. Skrypt łączący w jedną całość pliki *.txt w bieżącym katalogu.

 #!/bin/bash


   for plik in `ls *.txt`	 # lub $(ls *.txt) 
   do 
   cat $plik >> calosc.txt 
  done 

exit 0

5. Skrypt pyta o hasło i sprawdza czy jest poprawne . W tym skrypcie zastosowana jest funkcja warunkowa "if".

#!/bin/bash 


   echo "Podaj haslo" 

   read haslo 

   if [ "$haslo" = "tajne" ] 
   then 
   echo "Prawidlowe haslo" 
   else 
   echo "Dostep zabroniony" 
  fi 
  
exit 0

6.Skrypt ktory wyznacza silnie z podanego parametru.

#!/bin/bash


   i=1;
   wynik=1;

   while (( i<=$1)); 
   
  do
   wynik=$wynik*$i
   let "i++"
   done
   echo $wynik 
    
 done

exit 0
